## 📖 About FairSwarm
FairSwarm is an innovative **Swarm Intelligence AI Bias Detection Platform**. It combats algorithmic discrimination by deploying four distinct AI models simultaneously to audit the same dataset. By combining different perspectives (such as Llama 3, Gemini 2.0, Groq, and Mixtral), FairSwarm synthesizes a single, highly reliable **Fairness Score** using industry-standard algorithmic consensus metrics. 

## ✨ Key Features
* **🤖 Swarm Auditing Architecture:** A multi-model approach that guarantees a comprehensive and resilient audit against single-model blind spots.
* **📊 Deep Fairness Metrics:** Calculates true parity differences including *Disparate Impact Ratio*, *Statistical Parity Difference*, *Equal Opportunity Difference*, and more.
* **🚀 Blazing Fast:** Asynchronous processing enables most audits parallel execution to finish in under 5 minutes.
* **📱 Responsive & Interactive Data Viz:** Beautiful interactive charts leveraging Recharts and Framer Motion. 
* **☁️ Enterprise Serverless:** Hosted with full edge-capabilities on Vercel and Render.
